# Face Occlusion & Subliminal Priming Experiment

Experiment script for the bachelor's thesis *"Less is More Regardless of Priming:
Why Attractiveness-Gains by Occlusion are Robust to Subliminal Context"*.

Built in JavaScript and HTML using [lab.js](https://lab.js.org/). This script is not
standalone — it must be loaded and run within the lab.js environment.

## Dependencies
- [lab.js](https://github.com/felixhenninger/lab.js) (see their documentation for setup)

## Note on assets
The facial stimuli used in this study are from the FACES dataset (Ebner et al., 2010)
and are not publicly available. The images are therefore not included in this repository.

## Note on code
Parts of this script were developed with the assistance of generative AI tools.
